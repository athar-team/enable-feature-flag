enable-feature-flag plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/pomegranited/tutor-contrib-enable-feature-flag

Usage
-----

::

    tutor plugins enable enable-feature-flag
